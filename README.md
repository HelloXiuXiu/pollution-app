# Pollution-App
The Application shows the current air quality index along with
hourly rates for each pollutant.
<br>
This is a web version of the application. The Native App is build with [Socket runtime.](https://socketsupply.co/)
<br>
<br>
React + Vite. Run: 
<br>

```sh
npm i
npm run dev
```
